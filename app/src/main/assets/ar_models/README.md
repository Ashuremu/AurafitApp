# AR Clothing Images

This folder contains 2D images for AR clothing overlays.

## Folder Structure:
- `clothing/shirts/` - Shirt images
- `clothing/pants/` - Pants/shorts images  
- `clothing/shoes/` - Shoe images
- `clothing/accessories/` - Accessory images

## Image Requirements:
- **Format**: PNG (recommended for transparency)
- **Size**: 512x512px or 1024x1024px (square)
- **Style**: Front-facing view of clothing item
- **Background**: Transparent or solid color

## File Naming:
- `casual_tshirt.png` - Casual t-shirt
- `shorts.png` - Shorts
- `sneakers.png` - Sneakers
- `watch.png` - Watch

## How It Works:
1. Place PNG images in the appropriate folders
2. The AR system will automatically load and display them
3. If images are missing, colored rectangles will be shown as fallback
